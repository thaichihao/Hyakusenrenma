# Step 1

## Prerequisites

* [Node] and [NPM] (https://docs.npmjs.com/getting-started/installing-node) `8.2.1` or above
* [Gulp] (https://github.com/gulpjs/gulp/blob/master/docs/getting-started.md) 


## Install

Navigate to source base and run:

```shell
Gulp
```
